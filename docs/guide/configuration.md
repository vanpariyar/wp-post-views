# Configuration

The WP Post Views settings allow you to control which content is tracked and how the counting logic behaves. You can access these settings by navigating to **Settings > WP Post Views** in your WordPress dashboard.

## Settings Overview

![Settings Page](/screenshot-2.png)

### Count Views For
Select the post types you want to track. By default, **Posts** and **Pages** are selected. If you have custom post types (like Products or Portfolio), they will appear here as well.

### Exclude IP Addresses
If you want to prevent your own visits or bot traffic from skewing your view counts, you can enter IP addresses to exclude (one per line).

### Views Admin Column
Toggle whether to show the "Views" column in the WordPress admin post lists. This provides a quick overview of your most popular content.

## Gutenberg Block

WP Post Views comes with a modern Gutenberg block that you can use to display view counts anywhere in your content.

### Using the Block
1. In the Block Editor, click the **+** icon.
2. Search for **Post Views**.
3. Insert the block into your post or page.

![Gutenberg Block](/screenshot-3.png)
*The "Post Views" Gutenberg block in the block inserter.*

### Customizing the Block
You can customize the appearance of the block in the editor sidebar, such as changing the prefix text (e.g., "Total Views:") or styling.
