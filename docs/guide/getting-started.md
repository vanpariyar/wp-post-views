# Getting Started

WP Post Views is a lightweight and efficient plugin designed to track and display post views in WordPress with ease.

## Installation

You can install WP Post Views using one of the following methods:

### Via WordPress Admin
1. Go to **Plugins > Add New** in your WordPress dashboard.
2. Search for "WP Post Views".
3. Click **Install Now** and then **Activate**.

### Via FTP/Upload
1. Download the plugin from the [WordPress.org Plugin Directory](https://wordpress.org/plugins/wp-post-views/).
2. Extract the ZIP file.
3. Upload the `wp-post-views` folder to your `/wp-content/plugins/` directory.
4. Activate the plugin through the **Plugins** menu in WordPress.

## Initial Setup

Once activated, the plugin starts tracking views immediately for your posts and pages. You can see the view counts directly in your admin post list.
